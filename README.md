# pi-cicd

Extension for Pi coding agent.

## Install

### From npm (recommended)

```bash
pi install npm:pi-cicd
```

### From GitHub Packages

```bash
pi install npm:@baphuongna/pi-cicd
```

### From local source (development)

```bash
cd pi-cicd && pi install .
```

## Verify

```bash
pi list
```
